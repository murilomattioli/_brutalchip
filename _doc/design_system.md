# Design System

Libs:
- react-icons (https://react-icons.github.io/)
- daisyui (https://daisyui.com/)
- tailwindcss (https://tailwindcss.com/)


- Header 
  - Desktop: 
    - Logo
    - Search With Dropdown
    - [cart_page] (open modal with button for redirect to cart_page):
    - [user_page]:

  - Mobile:
    - Row 1:
      - Back Button  
      - Logo 
      - [cart_page] 
      - Menu Button
        - Categories
        - Promotion List
        - User Section:
          - Favorites
          - Orders
          - UserName -> [user_page]


[user_page] = 
  - Favorites
  - Personal Info
  - Addresses
  - Orders
  - History (bonus feature)

[cart_page] = 
  - Items
  - Subtotal
  - Address with delivery price
  - Discount
  - Total
  - Checkout

[checkout_page] =
- Resume (items, subtotal, delivery, discount, total)
- Payment
  - Credit Card
  - Debit Card
  - Boleto
  - Pix
- Order Confirmation
